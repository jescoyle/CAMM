Initialize simulation

Usage

initialize_camm(parm_file = NA, save_start = F, runID = NA, save_dir = "./")

Arguments

parm_file
path to file with parameters
save_start
logical indicating whether metacommunity objects should be saved to an .RData file. Defaults to FALSE.
runID
character string identifying this simulation. Required if save_start=TRUE.
save_dir
directory where simulation objects should be saved. Defaults to current directory.

Value

a named list containing all objects needed to run a simulation

Description

Initializes a simulation for a given set of parameters

Details

The function creates host and symbiont species pools, global species abundance distributions (gsad), an association network (and named links), an association probability matrix, a set of sites with environmental conditions, and an empty metacommunity at those sites. Parameters for creating these objects are read from a file (parm_file) or from the environmentwhere the function is called. Required parameters are:

S_a and S_b
number of host and symbiont species

N_L
number of links in the association network

topology
association network topology ('one2one', 'one2many' or 'many2many'). See make_topo.

N_C
number of sites

N
number of microsites/individuals per site

rho_z
correlation between environmental axes across sites

niche_parms_a and niche_parms_b
named list of parameters used to create host (a) and symbiont (b) niches. See make_niches_gsad.

gsad_dist_a and gsad_dist_b
named list of parameters used to create host (a) and symbiont (b) global species abundance distributions. See make_niches_gsad.

Optional parameters are:

comm_fill
character string indicating how initial metacommunity should be filled. Defaults to 'empty'. See link{make_comm}.
