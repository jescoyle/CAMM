Creation of an association network

Usage

make_topo(S_a, S_b, N_L, topology)

Arguments

S_a
(required) number of host species
S_b
(required) number of symbiont species
N_L
(required) number of edges (links) in the network
topology
(required) form of network topology. Can be 'one2one', 'one2many', 'many2many' (see details).

Value

a binary matrix with dimension S_a x S_b indicating links between host and symbiont species

Description

Creates an association network between a given number of host and symbiont species with a defined network topology.

Details

The function generates a random bipartite network between host and symbiont species which is returned as a matrix of 0s and 1s (indicating whether two species can associate with one another). Users can specify the network topology as:

'one2one'
each host associates with one symbiont and each symbiont associates with one host

'one2many'
each host associates with one symbiont and each symbiont associates with one or more hosts

'many2many'
each host associates with one or more symbionts and each symbiont associates with one or more hosts

The specified topology must be compatible with the number of species and number of links (N_L) specified. E.g. for 'one2one', the number of of host and symbiont species and links must be equal, whereas, for 'one2many' the number of links must be greater than the number of symbionts.
