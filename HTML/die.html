Death

Usage

die(comm, topo_names, pool, mortality, partner)

Arguments

comm
(required) metacommunity matrix indicating which association is present in microsite of each site. Rows are sites and columns are microsites/individuals.
topo_names
(required) matrix representing association network where integers label each association, as produced by name_topo. Labels should match those used in comm.
pool
(required) binary site x species matrix indicating which species are present in each site's species pool, but which are not necessarily established.
mortality
(required) probability that an unassociated mutalist dies
partner
(required) integer indicating whether the function applies to hosts (1) or symbionts (2).

Value

binary site x species matrix indicating which species remain in the species' pools at each site

Description

Random mortality of species in communities that are not established in a microsite as a mutualistic association.
