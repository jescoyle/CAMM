Community-environment correlations

Usage

calc_envcorr(comm, topo_names, env, metric = NULL, binary, pools = NULL)

Arguments

comm
(required) site x microsite matrix with integers representing the associations present in each microsite
topo_names
(required) host x symbiont association matrix with integers identifying each association (as returned by name_topo)
env
(required) site x env matrix with environmental conditions at each site
metric
vector of dissimilarity metric (see vegdist for options). Defaults to NULL for none.
binary
lvector of logicals indicating whether calculations should be performed on species abundances (FALSE, default) or presence/absence (TRUE) or both
pools
(required) named list of pools of hosts as symbionts present at each site. Pools are generated from the community matrix by calc_pool. The host pool should be named 'a' and the symbiont pool named 'b'.

Value

array with dimensions [community type, environmental axis, measurement, abundances vs presence/absence] (see details)

Description

Calculates multiple community-environment correlations

Details

This summary function calculates the correlation between each environmental axis (provided in env) and species richness, abundance and community structure for established associations and host and symbiont species involved in established associations. For each environmental axis and dissimilarity metric given in metric it calculates the correlation between a community dissimilarity matrix (see link{calc_diss}) and the environmental distance matrix. Finally, the function conducts a constrained ordination along each environmental axis (see link{calc_rda}) and returns the adjusted $R^2$. These statistics are returned as a 4-dimensional array where the first dimension indicates the community examined (associations: 'species', hosts: 'a', symbionts: 'b'), the second dimension indicates which environmental axis was used, the third dimension indicates the statistic (richness: 'S', abundance: 'N', RDA: 'rda', or correlation with community dissimilarity metric), and the fourth dimension indicates whether the calculations are based on abundances (FALSE or presence/absence (TRUE).
