Calculate community statistics

Usage

calc_commstats(comm, topo_names, pools)

Arguments

comm
(required) site x microsite matrix with integers representing the associations present in each microsite
topo_names
(required) host x symbiont association matrix with integers identifying each association (as returned by name_topo)
pools
(required) named list of pools of hosts as symbionts present at each site. Pools are generated from the community matrix by calc_pool. The host pool should be named 'a' and the symbiont pool named 'b'.

Value

a dataframe of community statistics (see details) with rows representing sites

Description

Calculate diversity and abundance of hosts, symbionts and associations

Details

This function calculates abundance and diversity of hosts, symbionts and associations that are established at sites. If a species is present in a site's species pool, but not in an association at a microsite it will not be counted. Three diversity statistics are calculated for hosts ('a'), symbionts ('b') and associations ('species'):

  • S : species richness (number of species present at each site), see calc_rich
  • Stot : total species richness across all sites, see calc_tot_rich
  • Beta : beta diversity (total richness divided by mean richness at each site)

The function also returns the number of microsites occupied at each site (N), the total number of microsites occupied across the entire metacommunity (Ntot), and the pearson correlation between species richness of hosts and symbionts across sites (Core_ab). Community statistics are returned as a dataframe with each row representing a site.
