Create parameter list

Usage

make_parmlist(e = parent.frame())

Arguments

e
environment where parameters are located

Value

a named list of paramters

Description

Creates a list of simulation parameters

Details

The function creastes a list of parameters required for running a simulation. Parameters are taken by default from the environment where the function is called, or optionally from the environment provided by e. Most parameters are required for simulation and must be present in the environment or the function will fail. Optional parameters are indicated.

Parameters for creating a metacommunity (see initialize_camm):

  • runID : name of this simulation run
  • S_a : number of host species
  • S_b : number of symbiont species
  • N_C : number of sites
  • N : number of microsites / individuals
  • rho_z : correlation between environmental axes across sites
  • N_L: number of links in the association network between host and symbiont species
  • topology: character string defining type of association network topology
  • N_L: number of links in the association network between host and symbiont species
  • comm_fill: (optional) whether initial metacommunity should be 'empty' or filled at 'random'

Parameters for initializing a metacommunity with data (see initialize_camm, all parameters are optional):

  • topo_data : path to csv file with species association matrix
  • site_data : path to csv file with site environmental data
  • gsad_a_data : path to txt file with host global species abundances
  • gsad_b_data : path to txt file with symbiont global species abundances

Parameters for creating species niches (see make_niches_gsad):

  • mu_a1 and mu_a2 : maximum niche optima for host species along environmental axes 1 and 2
  • mu_b1 and mu_b2 : maximum niche optima for symbiont species along environmental axes 1 and 2
  • rho_a and rho_b: correlation of niche optima between environmental axes across host and symbiont species
  • sigma_a1 and sigma_a2 : expected niche breadth of host species along environmental axes 1 and 2
  • sigma_b1 and sigma_b2 : expected niche breadth of symbiont species along environmental axes 1 and 2
  • alpha_a1 and alpha_a2 : shape parameter of the gamma distribution from which host niche breadths are drawn for environmental axes 1 and 2
  • alpha_b1 and alpha_b2 : shape parameter of the gamma distribution from which host niche breadths are drawn for environmental axes 1 and 2
  • r_a and r_b: correlation of niche breadths between environmental axes across host and symbiont species
  • gsad_dist_a and gsad_dist_b: list of paramters describing the global species abundance distribution for hosts ans symbionts

Parameters for running the simulation:

  • omega: degree of mutualism: 0 = no facilitation, 1 = obligate mutualism (see calc_probs)
  • assoc_prob_func: function that defines the probability that a host and symbiont will form an association, given that they have the capacity to. Should accept three arguments: env (vector of environmental conditions), niche_a (matrix defining the host's niche) and niche_b (matrix defining the symbiont's niche).
  • mort_rate: probability that an established association dies in a timestep
  • mort_rate_a and mort_rate_b: local extinction rate of unassociated hosts and symbionts from a species pool relative to the mortality rate of established associations (e.g., a value of 10 will cause cause unassociated mutualists to be removed from a species pool at 10x the rate of associated mutualists)
