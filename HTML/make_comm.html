Creation of metacommunity

Usage

make_comm(N_C, N, N_S = 0, fill = "empty")

Arguments

N_C
(required) number of communities (sites)
N
(required) number of microsites (or individuals) in each community
N_S
number of species pairs
fill
character string indicating how communities should be filled. Defaults to 'empty'. See details.

Description

Create a set of N_C communities, each with N microsites or individuals.

Details

A metacommunity is represented by a N_C x N matrix where rows give the host-symbiont pair present at each site. Integers in the matrix correspond to host-symbiont pairs named in the association network topology matrix. Currently implements two methods for starting a metacommunity which is specified by fill:

'empty'
initial metacommunity has no species present

'random'
initial metacommunity community populated by randomly sampling species pairs until it is full
