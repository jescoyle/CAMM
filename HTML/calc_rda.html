Community-environment RDA

Usage

calc_rda(comm, topo_names, type, env, binary = F, pool = NULL)

Arguments

comm
(required) site x microsite matrix with integers representing the associations present in each microsite
topo_names
(required) host x symbiont association matrix with integers identifying each association (as returned by name_topo)
type
(required) character string indicating whether the function should do calculations on host species ('a'), symbiont species ('b'), or associations ('species')
env
(required) matrix of environmental variables across sites
binary
logical indicating whether rda should be performed on species abundances (FALSE, default) or presence/absence (TRUE)
pool
site x species matrix for the species pool present at each site (as returned by calc_pool). Must be supplied if calculating for host or symbiont species.

Description

Calculate community ~ environment RDA

Details

The function calculates the correspondence between community composition and a given environmental data matrix (env) for established associations or host or symbiont species in established associations by performing a constrained ordination (rda) on Hellinger-transformed species abundances (as implemented by decostand). The function returns the adjusted eqnR^2 as a measured of correspondence between community structure and environment.
