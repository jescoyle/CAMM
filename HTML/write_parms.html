Write parameter file

Usage

write_parms(parm_list, file_name, file_dir = "./")

Arguments

parm_list
(required) named list of parameters to record
file_name
(required) path to file where parameters should be written
file_dir
directory where file should be saved, if not included in file_name. Defaults to current directory.

Value

nothing

Description

Write simulation parameters to a file

Details

This function writes paramater assignments from a named list of parameters to a given file. Each element in the list is written as a line that can be read into R (using source) such the name of the list element becomes the object name. The function only works for vectors, lists with one level, and functions. It then appends two lines defining niche parameter lists for hosts and symbionts.
