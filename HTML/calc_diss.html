Calculate community dissimilarity

Usage

calc_diss(comm, topo_names, type, metric, binary = F, pool = NULL)

Arguments

comm
(required) site x microsite matrix with integers representing the associations present in each microsite
topo_names
(required) host x symbiont association matrix with integers identifying each association (as returned by name_topo)
type
(required) character string indicating whether the function should do calculations on host species ('a'), symbiont species ('b'), or associations ('species')
metric
(required) dissimilarity metric used for calculation (see vegdist
binary
logical indicating whether dissimilarties should be calculated from species abundances (FALSE, default) or presence/absence (TRUE)
pool
site x species matrix for the species pool present at each site (as returned by calc_pool). Must be supplied if calculating for host or symbiont species.

Value

matrix of site dissimilarities, as returned by vegdist

Description

Calculate dissimilarity in species composition among sites

Details

This function uses vegdist in the vegan package to calculate community dissimilarity among sites. Dissimilarity can be calculated for associations or for host or symbiont species present in established associations. Any metric implemented by vegdist can be used and dissimilarities can be calculated based on abundances or based on presence/absence (binary=TRUE).
