Initialize simulation

Usage

initialize_camm(parm_file = NA, save_start = F, runID = NA, save_dir = "./")

Arguments

parm_file
path to file with parameters
save_start
logical indicating whether metacommunity objects should be saved to an .RData file. Defaults to FALSE.
runID
character string identifying this simulation. Required if save_start=TRUE.
save_dir
directory where simulation objects should be saved. Defaults to current directory.

Value

a named list containing all objects needed to run a simulation

Description

Initializes a simulation for a given set of parameters

Details

The function creates host and symbiont species pools, global species abundance distributions (gsad), an association network (and named links), an association probability matrix, a set of sites with environmental conditions, and an empty metacommunity at those sites. Parameters for creating these objects are read from a file (parm_file) or from the environment where the function is called. Required parameters are:

S_a and S_b
number of host and symbiont species

N_L
number of links in the association network

topology
association network topology ('one2one', 'one2many' or 'many2many'). See make_topo.

N_C
number of sites

N
number of microsites/individuals per site

rho_z
correlation between environmental axes across sites

niche_parms_a and niche_parms_b
named list of parameters used to create host (a) and symbiont (b) niches. See make_niches_gsad.

gsad_dist_a and gsad_dist_b
named list of parameters used to create host (a) and symbiont (b) global species abundance distributions. See make_niches_gsad.

Optional parameters are:

comm_fill
character string indicating how initial metacommunity should be filled. Defaults to 'empty'. See link{make_comm}.

topo_data
path to csv file with species association matrix. Path should be relative to the location of parm_file, if provided. File should contain a binary matrix without row or column names. If provided, overrides values for S_a, S_b, N_L, and topology.

site_data
path to csv file with site environmental data. Path should be relative to the location of parm_file, if provided. File should contain two columns: one for each environmental axis. If provided, overrides values for N_C and rho_z.

gsad_a_data
path to txt file with host global species abundances. Path should be relative to the location of parm_file, if provided. File should contain one line with species abundances (as integers) separated by white space. If provided, overrides values for gsad_dist_a and will not be correlated with niches.

gsad_b_data
path to txt file with symbiont global species abundances. Path should be relative to the location of parm_file, if provided. File should contain one line with species abundances (as integers) separated by white space. If provided, overrides values for gsad_dist_b and will not be correlated with niches.

See also

make_topo, make_niches_gsad, make_comm, make_sites
